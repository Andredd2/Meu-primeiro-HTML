O que é HTML

(Voltar)

HTML (HyperText Markup Language) é uma linguagem de marcação essencial e amplamente usada para estruturar e apresentar conteúdo na WWW. Criada por Tim Berners-Lee* no início dos anos 90, tornou-se a base para a criação de páginas web. O HTML organiza e dispõe diversos elementos de uma página da internet, como textos, imagens, links, tabelas, vídeos e outros componentes multimídia.

A estrutura básica de um documento HTML é formada por elementos, cada um com um papel específico na exibição da página. Esses elementos são delimitados por "tags" (marcadores), que são instruções entre os símbolos < e >. As tags orientam o navegador web sobre como interpretar, formatar e mostrar cada parte do conteúdo ao usuário.

Por exemplo, a tag < p > indica um parágrafo, < img > colocar insere uma imagem, < a > cria um link e < table > inicia uma tabela. Muitas tags podem ter atributos que fornecem informações extras sobre o elemento, como o atributo "src" em uma tag de imagem para indicar o caminho do arquivo. Essa estrutura hierárquica e semântica permite que os desenvolvedores criem páginas web complexas e interativas, oferecendo uma base sólida para a aplicação de estilos com CSS e funcionalidades dinâmicas com JavaScript.

O que é TAGS

(Voltar)

As tags HTML são elementos usados para marcar e estruturar o conteúdo de uma página web, indicando ao navegador como exibir textos, imagens, links e outros componentes. Cada tag é representada por uma tag de abertura (ex: < p >) e, na maioria dos casos, uma tag de fechamento (ex: < /p >), que delimitam o início e o fim do conteúdo. Existem tags específicas para diferentes tipos de conteúdo, como títulos (< h1 > a < h6 >), parágrafos (< p >), links (< a >), imagens (< img >) e tabelas (< table >).

Algumas tags, como < img > ou < br >, são auto-fechadas, ou seja, não precisam de uma tag de fechamento. As tags também podem ser semânticas, como < header >, < article >, e < footer >, ajudando a melhorar a acessibilidade e a otimização para mecanismos de busca (SEO).

Tags HTML são fundamentais para estruturar e exibir o conteúdo corretamente nas páginas web, criando a base para o desenvolvimento de qualquer site.

Tags: principais para iniciante

(Voltar)

Algumas tags HTML principais para iniciantes são:

  1. < html >: Define o início do documento HTML
  2. < head >: Contém links para scripts e estilos, além de metadados
  3. < body >: Contém todo o conteúdo visível da página
  4. < h1 > a < h6 >: Usadas para títulos e subtítulos, sendo que h1 é a mais importante e deve ser atribuída ao título principal
  5. < p >: Define um parágrafo
  6. < ul >: Lista não ordenada
  7. < ol >: Lista ordenada
  8. < i >: Tag de itálico
  9. < u >: Tag de sublinhado
  10. < strong >: dá ao texto uma forte importância, e é tipicamente mostrado em negrito
  11. < table >: Cria uma tabela para organizar dados em linhas e colunas
  12. < mark >: serve para destacar palavras-chaves ou outros trechos de texto
  13. < small >: é utilizado para reduzir o tamanho da fonte num intervalo definido de um texto
  14. < font >: é utilizado para configurar o tipo, cor e tamanho das letras de um texto de uma página HTML
  15. <.........>
